# QuickDoc v1.5.1

QuickDoc v1.5.1 fixes the PowerPoint template used when creating new `.pptx` files from Finder.

## Fixes

- Fixed an issue where newly created PowerPoint files could trigger a PowerPoint warning that the presentation content had problems and needed repair.
- Replaced the overly minimal built-in blank PPTX template with a complete PowerPoint-compatible template that includes slide master, slide layout, theme, document properties, notes parts, and relationship metadata.

## 中文说明

- 修复通过 Finder 右键新建 PowerPoint（`.pptx`）后，PowerPoint 可能提示文件内容有问题并要求修复的情况。
- 将过于精简的内置空白 PPTX 模板替换为更完整的 PowerPoint 兼容模板，补齐幻灯片母版、版式、主题、文档属性、备注部件和关系元数据。

## Downloads

- `QuickDoc-1.5.1.dmg`: recommended installer for manual installation.
- `QuickDoc-1.5.1.zip`: update package used by the in-app updater.

> Note: this release follows the current project signing workflow. If macOS warns that the app is from an unidentified developer, use Control-click > Open for the initial launch.
