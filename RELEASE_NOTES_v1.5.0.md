# QuickDoc v1.5.0

QuickDoc v1.5.0 adds custom third-party terminal support and refreshes the built-in icon system.

## Highlights

- Added custom terminal app support for `Open in Terminal`.
- Users can now choose a third-party terminal app from General settings instead of being limited to macOS Terminal.
- The selected terminal is shown with its app icon, name, and path, with direct actions to change the terminal app or restore the system Terminal.
- Refactored the built-in file type icons with a more unified visual style.

## 中文说明

- “在终端中打开”新增第三方自定义终端支持。
- 用户可以在“通用”设置页选择自己常用的第三方终端 App，不再局限于系统终端。
- 设置页会展示当前选定终端的图标、名称与路径，并提供“更换终端 App”和“恢复系统终端”两个操作。
- 内置文件类型 icons 完成统一化风格重构，整体视觉更一致。

## Downloads

- `QuickDoc-1.5.0.dmg`: recommended installer for manual installation.
- `QuickDoc-1.5.0.zip`: update package used by the in-app updater.

> Note: this release follows the current project signing workflow. If macOS warns that the app is from an unidentified developer, use Control-click > Open for the initial launch.
