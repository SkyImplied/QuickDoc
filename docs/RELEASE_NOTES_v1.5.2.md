# QuickDoc v1.5.2

QuickDoc v1.5.2 focuses on bug fixes, software usability refinements, and a more polished DMG installation experience.

## Highlights

- Fixed several small issues discovered in daily use.
- Refined the app experience and refreshed the About page with the latest feature overview and contact information.
- Improved the DMG installer layout with a clearer drag-to-Applications flow, larger icons, centered content, and sharper installation text on Retina displays.

## 中文说明

- 修复一些已知问题。
- 优化软件使用体验，并更新“关于”页面中的功能介绍与联系信息。
- 优化通过 DMG 安装的软件安装流程，安装窗口现在包含更清晰的拖拽指引、更大的图标、居中的内容布局，以及更清晰的 Retina 安装文案。

## Downloads

- `QuickDoc-1.5.2.dmg`: recommended installer for manual installation.
- `QuickDoc-1.5.2.zip`: update package used by the in-app updater.

> Note: this release follows the current project signing workflow. If macOS warns that the app is from an unidentified developer, use Control-click > Open for the initial launch.
