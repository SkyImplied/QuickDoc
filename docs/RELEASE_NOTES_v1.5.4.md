# QuickDoc v1.5.4

QuickDoc v1.5.4 adds custom file templates, making Finder file creation much closer to real daily document workflows.

## Highlights

- Added custom file templates for enabled file types, including built-in types such as Word, Excel, PowerPoint, TXT, and user-defined custom extensions.
- Added a redesigned template manager that groups templates by file type and keeps each type's upload, display, ordering, and visibility settings separate.
- Added custom template display names, visibility toggles, reordering controls, rename support, and deletion.
- Finder's `New File` menu now opens a third-level submenu for file types with visible templates. The first option remains `Blank default document`, followed by the user's templates in the configured order.
- Template file types now follow the enabled new-file types from settings, and custom file types use the unified blank-file icon.
- Fixed the extra blank separator in the third-level template submenu.

## 中文说明

- 新增自定义文件模板功能，可为已启用的文件类型上传自己的模板文档，包括 Word、Excel、PowerPoint、TXT 以及用户自定义后缀。
- 新增重新设计的模板管理板块，按文件类型分组管理，每个类型都有独立的上传、显示、排序和隐藏设置。
- 支持自定义右键菜单显示名、显示/隐藏开关、排序调整、重命名和删除。
- Finder 右键的“新建文件”菜单在某个文件类型存在可显示模板时，会自动进入三级菜单；第一个选项保留“空白默认文档”，后面按设置顺序显示用户模板。
- 模板类型选择会跟随上方已启用的新建文件类型，自定义文件类型统一使用空白文件图标。
- 修复模板三级菜单中多出一行空白间隔的问题。
